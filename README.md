## Install

To install ariake.nvim you need a plugin manager.

- [Lazy.nvim](https://github.com/folke/lazy.nvim)

Example with Lazy.nvim

```lua
return {
    {
        'jim-at-jibba/ariake.nvim',
        config = function ()

         vim.cmd.colorscheme 'ariake'
        end
    }
}
```
